# MySQL Practice

This directory contains SQL scripts and notes for practicing MySQL features, specifically focusing on **Partitioning**.

## Partitioning

### Goals
- Implement Range Partitioning
- Implement List Partitioning
- Implement Hash Partitioning
- Practice partition management (ADD, DROP, REORGANIZE)
