# PostgreSQL Practice

This directory contains SQL scripts and notes for practicing PostgreSQL features, specifically focusing on **Partitioning**.

## Partitioning

### Goals
- Implement Declarative Partitioning (Range, List, Hash)
- Practice partition management (Attach, Detach)
- Explore Partition Pruning
