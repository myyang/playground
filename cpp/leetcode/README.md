# LeetCode

LeetCode algorithm problems implemented in C++ with unit tests.

## Structure

```
├── src/           # Solution implementations
│   ├── dp.cpp         # Dynamic programming problems
│   ├── binary_search.cpp
│   ├── two_pointers.cpp
│   ├── graph.cpp
│   ├── search.cpp
│   ├── trie.h
│   └── adv.cpp        # Advanced topics
└── test/          # Google Test unit tests
```

## Build & Run

```bash
mkdir build && cd build
cmake ..
make
./tests
```

## Topics Covered

- Dynamic Programming (climbing stairs, tribonacci, range sum, etc.)
- Binary Search
- Two Pointers
- Graph Algorithms
- Trie Data Structure
- Search Algorithms
